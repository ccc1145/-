# AI Agent 框架示例文档

## RAG 介绍

检索增强生成（Retrieval-Augmented Generation, RAG）是一种结合外部知识库与大语言模型的技术。
其核心流程为：先从知识库中检索与用户问题相关的文档片段，再将这些片段作为上下文输入给 LLM，
从而生成有据可依的回答，显著降低幻觉。

本框架的 RAG 模块包含五个组件：

1. **文档加载器**：将 PDF / TXT / Markdown 解析为统一 Document 结构
2. **文本分块器**：按可配置策略将长文档切分为适合检索的片段
3. **嵌入模型**：将文本映射为高维向量
4. **向量存储**：持久化向量并支持相似度搜索
5. **检索器**：向量检索或 BM25+向量混合检索（RRF 融合）

## 框架特性

- 模块化：核心控制 / RAG / 知识库 / 工具 / 配置 / 交互 六大模块
- 可扩展：插件机制 + 协议抽象 + 注册中心
- 可配置：YAML + 环境变量，支持多 LLM/Embedding provider
- 工具调用：Agent 支持 ReAct 风格的工具决策循环

## 适用场景

可作为智能客服、企业知识问答、研究助手、文档分析等具体 Agent 应用的基座。
